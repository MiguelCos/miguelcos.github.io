# AGENTS.md

Guidance for future Codex tasks in this repository.

- Keep the site static, lightweight, and maintainable.
- Use Quarto source files (`.qmd`) and human-readable content.
- Do not add a backend, contact form, analytics, or heavy JavaScript framework unless explicitly requested.
- Do not publish sensitive personal data such as phone number, nationality, immigration or residency status, private address, or private CV details.
- Keep public CV content professional and privacy-conscious.
- Do not fabricate publication metadata, repository metadata, roles, awards, or affiliations.
- Prefer verified public sources for publication and repository updates.
- Run `quarto render` before finalizing when Quarto is available.
- If Quarto is unavailable, still check links and source files as far as possible and state that render was not run.
- Preserve existing content unless it is clearly superseded by the requested change.
